# E-commerce Reviews Sentiment Analysis

## Overview
This project focuses on sentiment analysis of Malaysian e-commerce product reviews collected from Shopee Malaysia and Lazada Malaysia.

## Objectives
- Collect and analyze Malaysian e-commerce product reviews.
- Perform text preprocessing and sentiment labeling.
- Train machine learning and deep learning models for sentiment classification.
- Compare model performance using standard evaluation metrics.
- Deploy a web-based sentiment prediction system.

---

## Dataset

Data sources:
- Shopee Malaysia
- Lazada Malaysia

The dataset consists of reviews from multiple product categories, including:
- Electronics Accessories
- Beauty & Skincare
- Food Products
- Fashion Apparel
  
Collected attributes:
| Column | Description |
|----------|------------|
| review_id | Unique review identifier |
| platform | Shopee / Lazada |
| category | Product category |
| product_name | Product name |
| rating | User rating |
| review_text | Customer review |
| sentiment | Positive / Neutral / Negative |
